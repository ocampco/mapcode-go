# :japan: Mapcode GO - Driving through Japan made easy

When driving through Japan, it's much easier to use car navigation using a [Denso Mapcode](https://en.wikipedia.org/wiki/Denso_mapcode) instead of manually entering location.

Getting the mapcode was difficult as the existing websites are unreliable and don't work well on different device sizes.

Mapcode GO is simple. Copy the Google Maps link for where you want to go and you'll recieve a map code you can enter into your car's navigation.
